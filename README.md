# API-interface-based-on-HTTP
Develop an API interface based on an HTTP service to dynamically and randomly return images from a directory on the server.

(1) Design wo API interfaces: one for randomly returning images and one for randomly downloading images.

(2) Install a Tomcat server and write a servlet program to handle HTTP requests and implement the required functions.

(3) Configure a JAVA environment and Tomcat, and package and deploy the entire project to the server.
